# 🚀 Big-data-learning-environment-linux-hadoop

Repositorio para instalación y configuración de:

- 🖥️ VirtualBox
- 🐧 Ubuntu
- 🐘 Hadoop
- 🐝 Hive
- ⚡ Spark

Además incluye:

- 🤖 scripts de automatización
- 📊 proyectos de Big Data
- 📓 notebooks
- 🗂️ datasets
- 📚 documentación técnica

---

# 🏗️ Arquitectura

🧩 Single Node Hadoop Cluster

---

# 🛠️ Tecnologías

- 🐘 Hadoop 3.4.1
- 🐝 Hive 4.0.1
- ⚡ Spark 3.5.3
- 🐧 Ubuntu
- ☕ Java 11

---

# 🎯 Objetivos del Proyecto

- 📦 Aprender ecosistemas Big Data
- ⚙️ Automatizar instalaciones
- 🧪 Realizar pruebas locales
- 📈 Desarrollar proyectos de Data Engineering
- 🔍 Experimentar con procesamiento distribuido

---
